# 材料与叠层

## 1. 材料与EMC性能优化

- **合适材料选择**：选用合适的PCB材料和厚度，以减少阻抗，改善电磁兼容性（EMC）性能。
- **EMC测试**：设计完成后进行EMC测试，检查干扰问题，并及时修复。这相当于房屋建成后的验收，确保每个部分都符合标准，没有隐患。

## 2. PCB 制造工艺

- **拼板与工艺边管理**：
  - 拼板：将多个小型PCB设计合并到一个大板上进行制造。
  - 工艺边（工艺边缘或工艺边界）：在加工、切割、钻孔等操作中保留至少0.5mm的边距，防止功能区域受损。

- **常见制造工艺**：
  - V割和邮票孔（邮票孔工艺）：用于切割特定形状的电路板，通常在拼板过程中使用。
  
- **MARK点**：用于标记元件精确放置位置，并作为测试点。有助于快速定位组装过程中的问题。
