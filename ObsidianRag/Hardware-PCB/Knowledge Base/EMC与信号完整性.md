# EMC与信号完整性

## 5. 散热与布线策略

- **散热考虑**：
  - 将高功耗元件放得远一点，以避免对其他元件的热影响。
  - 在散热区域留出空隙，方便空气流通。

- **元件尺寸**：
  - 合理利用空间，选取适合尺寸的元件以避免过拥挤。
  
- **信号线规则**：
  - 短而直的信号线和足够的电源线长度减少损耗。
  - 数码信号和模拟信号分开布置以避免干扰。

- **拐角设计**：使用圆弧或曲线走线方式，避免锐角和直角的问题。

- **3W与10W规则**：
  - 线中心间距不少于3倍线宽以减少信号串扰。
  - 满足10W原则可以将串扰降低近98%。
