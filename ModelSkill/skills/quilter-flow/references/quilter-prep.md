# Quilter 上传前准备 / 回收对比清单

## 上传前（在 KiCad 完成）
- [ ] 原理图完成，**ERC 干净**（用 `kicad-mfg` 跑 `run_erc`）。
- [ ] 所有元件有**正确封装**与 3D（避免占位/高度冲突）。
- [ ] **板框 Edge.Cuts** 闭合，外形/尺寸/圆角确定。
- [ ] 安装孔、定位孔、固定连接器位置确定并**锁定**。
- [ ] 预放置：USB/排针/屏蔽罩/天线/拨码等"机械约束件"摆好。
- [ ] 叠层定义：层数(2/4/6…)、铜厚、介质（高速/阻抗需要）。
- [ ] 网类(net class)：电源、GND、差分对(USB/以太/HDMI)、高速时钟，设线宽/间距/阻抗。
- [ ] 禁布区(keepout)：天线净空、连接器周边、散热/螺丝孔区。
- [ ] 标注"**关键 20%**"：高速总线/敏感模拟/电源分配——决定是否自己预布线。

## 混合流程（推荐）
- 自己预布关键 20%（DDR/USB/电源平面），锁定；其余 80% 交 Quilter。

## 回收后对比（多候选择优）
| 指标 | 看什么 |
|------|--------|
| 走线总长 / 关键网长度 | 越短越好，高速网看等长/对称 |
| 过孔数 | 越少越好（成本/可靠性） |
| 层使用 | 是否多用了层、平面完整性 |
| 间距裕量 | DRC 余量、是否贴极限 |
| 回流路径 | 高速网参考平面是否连续(EMC) |

## 选定后
- 回 KiCad **必跑 DRC**（`kicad-mfg` 的 `run_drc`），收尾丝印/测试点。
- 再导出制造文件（Gerber/钻孔/贴片/BOM）。

## 提醒
- Quilter 为网页/文件流程，无法本地脚本自动上传下载——按官网 workflow 操作。
