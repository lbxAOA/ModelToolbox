# 外设配置清单（通用五步 + 易错点）

通用五步：**开时钟 → 配引脚复用 → 配参数 → 配中断/DMA → 使能**。

## GPIO
- 模式：输入/输出/复用/模拟；输出类型推挽 vs 开漏；速度；上下拉。
- 易错：忘开端口时钟；开漏输出驱动外设需外部上拉。

## UART / USART
- 参数：波特率、数据位、停止位、校验、过采样。
- 收发：轮询 / 中断(RXNE/TXE) / DMA。高速或连续流用 DMA + 空闲中断(IDLE)。
- 易错：波特率误差(时钟源/分频)；TX 未等发送完成就改缓冲。

## SPI
- 参数：主/从、CPOL/CPHA(模式0~3)、波特率分频、位序(MSB/LSB)、NSS 软/硬。
- 易错：CPOL/CPHA 与从设备不匹配；片选时序；全双工读必须先写。

## I2C
- 参数：标准/快速模式、时钟、7/10 位地址、上拉电阻(必需)。
- 易错：缺上拉；总线死锁(从机拉死 SDA)需九脉冲恢复；ACK/NACK 处理。

## ADC
- 参数：分辨率、采样时间、单次/连续/扫描、触发源、对齐、参考电压。
- 多通道/高速：DMA 循环搬运；注意采样保持时间与源阻抗匹配。

## Timer / PWM
- 计算：`f_pwm = f_tim / ((PSC+1)*(ARR+1))`；占空比 = CCR/(ARR+1)。
- 模式：基本定时/输出比较/PWM/输入捕获/编码器。
- 易错：PSC/ARR 溢出；通道 GPIO 复用没配。

## DMA
- 配置：方向、外设/内存地址、增量、数据宽度、循环/单次、优先级、FIFO。
- 中断：传输完成(TC)、半传输(HT)、错误(TE)。
- 易错：内存对齐/宽度不一致；外设侧 DMA 使能位没开；cache 一致性(带 cache 的核)。

## 时钟树
- 选源(HSE/HSI/PLL) → 配 PLL 倍频 → 设 AHB/APB 分频 → 切换系统时钟 → 更新 SystemCoreClock。
- 易错：超频/总线超规格；外设时钟来自 APB1/APB2 不同；Flash 等待周期没设够。

## 中断 / NVIC
- 使能外设中断源 → 设优先级分组与优先级 → NVIC 使能 → 写 ISR 清标志。
- 易错：忘清中断标志导致反复进入；优先级反转；ISR 太长。
