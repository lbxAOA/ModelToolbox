# 建模方法速查（按问题类型）

挑选时考虑：数据量与质量、是否要可解释性、求解复杂度、比赛/审稿习惯。

## 优化 / 规划
- 线性规划 LP、整数规划 IP/MILP、非线性 NLP、多目标(加权/ε约束/Pareto)。
- 启发式：遗传算法、模拟退火、粒子群、蚁群；大规模/NP 难时用。
- 工具：Python `scipy.optimize`、`pulp`、`cvxpy`、`gurobipy`（如可用）。

## 预测 / 拟合
- 回归(线性/多项式/岭/LASSO)、时间序列(ARIMA/指数平滑)、灰色 GM(1,1)（小样本）、
  机器学习(随机森林/XGBoost/神经网络)。
- 注意：过拟合、训练/验证划分、误差指标(RMSE/MAE/MAPE/R²)。

## 评价 / 决策
- 层次分析 AHP（一致性检验 CR<0.1）、TOPSIS、熵权法、模糊综合评价、
  数据包络 DEA、主成分 PCA 降维。

## 图与网络
- 最短路(Dijkstra/Floyd)、最小生成树、最大流/最小割、匹配、网络流、社区发现。

## 连续 / 动力学
- 常微分方程组、偏微分方程(扩散/波动)、元胞自动机、传染病 SIR/SEIR、
  系统动力学（库存-流量）。

## 随机 / 统计
- 蒙特卡洛模拟、马尔可夫链/排队论、假设检验、方差分析、贝叶斯。

## 灵敏度 / 稳健性（必做）
- 单参数扰动曲线、Tornado 图、Sobol 全局灵敏度、对噪声/缺失数据的鲁棒性。

## 数模评审雷区
- 假设不给理由；符号前后不一致；只有结果没有检验；模型与题目数据脱节；
  附录代码无法复现。
